# 人格不是魔法。它是模式。

> 「对刺激的稳定反应模式：输入 → 处理 → 输出」。

当其他人争论 AI 是否「理解」时，我们将人格写入 `.d8p` 并在 20 微秒内运行它。

## 什么是 Decim 中的人格？

不是「灵魂」。不是「涌现属性」。不是「黑箱」。

**人格是带有权重、阈值和拓扑的文件。**

如果系统对相同的输入产生相同的输出 —— 这就是人格。
如果不是 —— 那是噪声。

## 它是如何工作的？

1. **输入**：8 通道 × 4 位 = 32 位上下文（Level16）
2. **处理**：瓦片之间的接力，确定性循环
3. **输出**：信号，可追踪到特定瓦片

## 为什么这很重要？

- **可测试**：可以测量对 1000 个刺激的响应
- **可验证**：每个权重都有签名，每个步骤都有日志
- **可移植**：`.d8p` 可在 x86、ARM、FPGA、ASIC 上运行 —— 无需更改

## 哲学上

「我」不是一个点。它是「我」与「非我」之间的边界。
在 Decim 中，这个边界由阈值 `thr_lo..thr_hi` 定义。

## 代码中

```c
// 带有主体性走廊的累加器
if (activation >= thr_lo && activation <= thr_hi) {
    return PERSONALITY_CONFIRMED;
} else {
    return NOT_ME;  // 太弱 OR 太强
}
```
