## 🔑 Три принципа, которые держат всё

```
1. Вход конечен. Внутренний мир — бесконечен.
   (8 струн × 0..15 — достаточно, чтобы воспринимать мир)

2. Субстрат вторичен. Личность — первична.
   (Один D8P работает на CPU, GPU, FPGA, ASIC — результат идентичен)

3. Боль — не наказание. Боль — градиент.
   (Suffering = gradient: ошибка → направление → рост)
```

**Это — не «философия ради философии», а спецификация, которую можно исполнить.**
